# DocSpot - Hospital Management System

DocSpot is a full-stack healthcare management system built using Django and React.  
It helps manage hospital-related workflows such as users, doctors, appointments, schedules, and admin-side operations.

## About the Project

This project was developed as a major academic project to simplify hospital management activities through a web-based platform.

## Tech Stack

- Frontend: React, TypeScript, JavaScript, HTML, CSS
- Backend: Django, Python
- Database: SQLite
- Tools: Git, GitHub, VS Code

## Key Features

- User/patient management
- Doctor management
- Appointment handling
- Doctor weekly schedule management
- Admin-side data control
- Responsive frontend interface
